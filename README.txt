Использовался плагин "1С 7.7 Exchange" для OpenCart 1.5.x.
https://opencartforum.com/files/file/2300-%D0%BE%D0%B1%D0%BC%D0%B5%D0%BD-1%D1%81-77-%D1%82%D0%BE%D1%80%D0%B3%D0%BE%D0%B2%D0%BB%D1%8F-%D0%B8-%D1%81%D0%BA%D0%BB%D0%B0%D0%B4-%D1%81-opencart-ocstore/
Произвел адаптацию под версию OC 2.0.3.1

Возможности:
 - Обмен с 1С 7.7 Торговля и склад OC 2.0.3.1;
 - В настройках плагина имеется возможность автоматического добавления нужного количество товара в указанную категорию;
    (ID категории менять в файле admin/model/tool/exchange.php на 368 строчке).

Установка:
 - Скопировать в рут директорию на сервере;
 - ЭкспортВOpenCart.ert копируем в папку ExtForms базы 1с.
